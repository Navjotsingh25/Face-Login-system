Face login system

Facial recognition system which can be intergrated on a website or an application and can be used by a person to login into a system. Facial recognition is achieved by OpenCV's face_recognition wrapper function and the sigup/login pages are built using HTML and CSS. Database of image encodings are stored in Django's database.
